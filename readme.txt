Enabling Latitude Interest Free Payment Gateway on a WooCommerce store.

== Description ==

The Latitude Checkout provides a simple and secure integration for WooCommerce Merchant partners to control their products that are allowed on Interest Free plans, and to accept the instant Interest Free payments from their Customers. The Customer’s payment journey will leverage Latitude’s PCI-compliant hosted payment page.

== Installation ==

This section outlines the steps to install the Laitude Interest Free plugin.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
